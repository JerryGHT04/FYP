# README: Charge-Exchange Cross-Section Data

This folder contains digitized charge-transfer cross-section data from classic measurements by Stebbings et al. (1963–1964).

## Sources

1. **Stebbings, R. F., Turner, B. R., Smith, A. C. H.**  
   *Charge Transfer in Oxygen, Nitrogen, and Nitric Oxide*  
   *J. Chem. Phys.*, 38(9), 2277–2279 (1963). DOI: 10.1063/1.1733961  
   - Systems:  
     O₂ + (O₂⁺, N⁺, O⁺, N₂⁺, NO⁺)  
     N₂ + (O₂⁺, N⁺, O⁺, N₂⁺, NO⁺)  
     NO + (NO⁺, N₂⁺)

2. **Stebbings, R. F., Smith, A. C. H., Gilbody, H. B.**  
   *Charge Transfer between Some Atmospheric Ions and Atomic Oxygen*  
   *J. Chem. Phys.*, 38(9), 2280–2284 (1963). DOI: 10.1063/1.1733962  
   - Systems: O + (N⁺, N₂⁺, NO⁺, O₂⁺)

3. **Stebbings, R. F., Smith, A. C. H., Ehrhardt, H.**  
   *Charge Transfer between Oxygen Atoms and O⁺ and H⁺ Ions*  
   *J. Geophys. Res.*, 69(11), 2349–2355 (1964). DOI: 10.1029/JZ069i011p02349  
   - Systems: O + (O⁺, H⁺)

## Data Format
- Files: `Energy(eV)  CrossSection(m²)`  
- Columns separated by two spaces.
